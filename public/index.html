English UEB Braille Translator (WASM)

About this Project

This project is a pure Python port of the world-renowned Liblouis Braille translation engine, developed by the power_python team. By converting the original C logic into Python and deploying it via WebAssembly (Pyodide), we bring industry-standard Braille translation directly to the web without the need for server-side processing or complex C-based installations.

Our Mission

Our goal is to provide a portable, accessible, and high-performance translation tool for Unified English Braille (UEB). This implementation maintains strict parity with Liblouis, ensuring that users receive the same high-quality Braille output they trust, anywhere Python or a modern web browser is available.

Usage Guide

  • Live Forward Translation: Enter standard text in the top box to see its Braille equivalent instantly.
  • Live Backward Translation: Enter Braille dots or BRF patterns in the bottom box to convert them back to readable text.
  • Switching Grades: Use the radio buttons to toggle between UEB Grade 1 (uncontracted) and UEB Grade 2 (contracted).
  • File Processing: Use the "Upload" button to process .txt or .brf files.
  • Export: Download your results using the provided "Download" buttons.
